# SpeedrunApp
Application using the www.speedrun.com API.
App done with:
* Kotlin
* Clean architecture + MVP
* Dagger2
* Retrofit2
* RxJava2, RxAndroid, RxKotlin
* Room
* Unit tests with Mockito and MockitoKotlin
* Instrumented tests with MockWebServer
